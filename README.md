## v6.1

#### 2026.4.19

本次主要对前端交互和整体架构进行了大幅优化，将原本的 PyQt5 版本全面迁移为 Web 版本，优化用户体验与系统性能

3，对系统进行全面测试，修复事件绑定、坐标偏移、缩放失效、数据更新异常等多个问题
